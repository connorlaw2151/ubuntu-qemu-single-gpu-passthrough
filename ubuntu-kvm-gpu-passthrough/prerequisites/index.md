## Passthrough of a Single GPU for use on a VM - Ubuntu
# Prerequsites
For this tutorial, you should have a working understanding of the linux terminal, previous experiance configuring applications in your terminal will be of assistance, a decent understanding of the working of workings of the linux kernal and software on your system is a plus, and an understanding of virtualization software is a plush.

Before you begin you should make sure that you have:
+ Installed the Linux Distribution you will be using for this tutorial
+ Updated your distribution
```
sudo apt update
sudo apt upgrade
```
+ Install the most updated graphics drivers for your system.  
For Ubuntu, you can type press the superkey and type `Additional Drivers`, launching the corresponding program to assist you in the installation of appropriate drivers
